(x, y, z) is the primary key column for this table.

Problem Statement

Report for every three line segments whether they can form a triangle.
Return the result table in any order.

+----+----+----+----------+
| x  | y  | z  | triangle |
+----+----+----+----------+
| 13 | 15 | 30 | No       |
| 10 | 20 | 15 | Yes      |
+----+----+----+----------+

ðŸ’¡ Explanation

A set of three sides (x, y, z) can form a valid triangle if and only if:

x + y > z
x + z > y
y + z > x


If all three conditions are true â†’ output 'Yes',
otherwise â†’ output 'No'.

âœ… SQL Solution
SELECT 
    x, 
    y, 
    z,
    CASE 
        WHEN x + y > z AND x + z > y AND y + z > x THEN 'Yes'
        ELSE 'No'
    END AS triangle
FROM Triangle;
